# Tweet_Analyzer_Demo
Python script that analyzes tweets. Counts frequency of common words in searched tweet.

# Requirements
# Libraries 
(1) Tweepy (For accessing Twitter API)
(2) pandas
(3) nltk
(4) nltk.corpus 
(5) collections
(6) matplotlib

# Twitter API
(1) api_key
(2) api_secret

(3) access_token
(4) access_token_secret

(5) application for api at https://developer.twitter.com/en
